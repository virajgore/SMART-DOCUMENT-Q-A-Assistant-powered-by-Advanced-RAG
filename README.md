# SMART DOCUMENT Q&A Assistant (Advanced RAG)

A **multi-document, conversational Retrieval-Augmented Generation (RAG) system** built using **FAISS, LangChain, Groq LLM, and AWS S3**.

This project allows users to:
- Upload PDF documents
- Persist vector embeddings in S3 (no re-embedding for the same document)
- Select one or multiple previously indexed documents
- Ask conversational questions across documents
